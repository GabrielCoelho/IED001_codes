## Filas Simples

- Vetor de inteiros.
- Precisamos saber o início e o fim da fila.
 	- Quando a fila anda, a variável de início é iterada, enquanto quando chega outro item na fila, a variável de fim é iterada.
 	- Estas variáveis iniciam no mesmo lugar, pois a fila está vazia. Logo: quando o início é igual ao fim, a fila está vazia.
 	- Para saber quando a fila está cheia, basta verificar se o fim é igual ao tamanho.

Vamos desenvolver uma fila simples.
